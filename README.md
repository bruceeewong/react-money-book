### 月份选择器

需求

- 按钮显示传入的年月，点击按钮可以打开和关闭下拉菜单
- 显示前后四年和 01月到12月
- 传入的年和月，打开菜单以后应该高亮
- 点击不同的年可以切换，点击月份应该触发回调

分段开发

- 创建一个按钮，显示传入的年和月份
- 创建一个下拉菜单，点击按钮可以显示，再点击隐藏
- 在下拉菜单中，显示两列，分别为年份和月份的信息
- 给这两列添加对应的选择高亮
- 添加点击年份和月份后的交互
